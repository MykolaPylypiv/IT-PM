# My Website

## Опис проєкту
Це простий фронтенд-сайт для практики тестування та QA.  
Він містить:
- Головну сторінку з кнопкою  
- Сторінку "About"  
- Сторінку "Contact" з формою і валідацією  

Проєкт створений як **MVP (Minimum Viable Product)** для демонстрації базового HTML/CSS/JS та перевірки основних функцій.

---

## Ідея MVP
Мета MVP — надати простий, стабільний сайт, який:
- працює на будь-якому браузері
- дозволяє тестувати навігацію, форми та інтерактивні елементи
- має мінімальний дизайн для зручного тестування
- не містить багів

Цей сайт **не призначений для комерційного використання**, тільки для навчання та QA-практики.
